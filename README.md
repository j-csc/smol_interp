# Autosteer

Mechanistic interpretability with Sparse Autoencoders - explore and steer language model behavior.

## Quick Start

```bash
# Setup (first time)
./setup.sh

# Run development servers
./run.sh
```

Frontend: http://localhost:3000

## Project Structure

- `backend/` - Modal serverless GPU backend (Python)
- `frontend/` - Next.js web application (TypeScript/React)

See [CLAUDE.md](./CLAUDE.md) for detailed development documentation.
